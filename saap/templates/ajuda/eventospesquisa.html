Pesquisa de eventos


Ao acessar a tela de Eventos, é possível, além de adicionar um novo evento, fazer uma pesquisa no cadastro de eventos.

Campos para filtro

Ainda não há campos para filtro. No futuro, serão disponibilizados.

Tabela de eventos

Todos os eventos são exibidos em uma tabela. O Total de Eventos aparece no alto da tabela.

Porém, para não sobrecarregar a exibição, quando há mais de 100 eventos, eles são divididos em páginas. É possível avançar para a página Anterior ou Próxima, assim como ir direto para uma página, clicando no número. A página atual é destacada com fundo azul claro.

Na tabela, são exibidas quatro colunas: Título, Bairro/Município, Início e Término. Para ordenar os contatos por uma destas colunas, basta clicar em cima dela. Quando há uma seta para cima ao lado do nome, significa que a ordem é crescente ou A-Z. Quando a seta aponta para baixo, a ordem é decrescente ou Z-A. Lembre-se que se a ordem é crescente, os dados em branco aparecem primeiro, e se a ordem é descrescente, os dados em branco aparecem por último.

Para acessar os dados de um evento, basta clicar no seu título (coluna Título, em verde).
